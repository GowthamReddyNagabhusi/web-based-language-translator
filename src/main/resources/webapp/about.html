🌐 About Translator v2.0

A production-ready, full-stack language translator delivering instant translations across 8 languages with auto-detection, pronunciation guides, keyboard shortcuts, and persistent history — all from a single deployable JAR.

✨ Features

  • Live translation — real-time as-you-type with configurable debounce
  • 8 target languages — French, Spanish, German, Hindi, Japanese, Chinese, Italian, Telugu
  • Pronunciation / romanization — for Japanese, Hindi, and Telugu scripts
  • Text-to-speech — listen to translations via Web Speech API
  • Share translations — via Web Share API or clipboard fallback
  • Translation history — search, filter, export/import as JSON, favorite items
  • Keyboard shortcuts — Ctrl+Enter, Esc, Ctrl+Shift+C, Ctrl+Shift+S
  • Word count & reading time — live stats as you type
  • Dark mode — system preference detection + manual toggle, persisted across sessions
  • Responsive design — optimized for mobile, tablet, and desktop

🏗️ Architecture

  • Backend: Java 17 + Embedded Jetty 11 (no framework overhead)
  • API: POST /translate — accepts JSON, returns structured response
  • Translation chain: Google Translate → LibreTranslate → MyMemory (cascading fallback)
  • Health endpoint: GET /health for container orchestrators
  • Frontend: Vanilla ES Modules — zero build step, zero dependencies
  • Logging: SLF4J + Logback with structured output

🛠️ Technologies

  • Java 17, Maven, Embedded Jetty 11, Jakarta Servlets
  • Gson for JSON serialization
  • SLF4J + Logback for production logging
  • Vanilla JavaScript (ES Modules), Fetch API, AbortController, Web Share API
  • CSS Grid, Flexbox, CSS Custom Properties, CSS Animations
  • Docker multi-stage build with health checks

⌨️ Keyboard Shortcuts

  • Ctrl+Enter — Translate immediately
  • Esc — Clear input and output
  • Ctrl+Shift+C — Copy translation to clipboard
  • Ctrl+Shift+S — Share translation

👨‍💻 Developer
