About Linguify

A production-grade, full-stack language translator delivering instant translations across 8 languages with auto-detection, pronunciation guides, keyboard shortcuts, and persistent history.

Features

  • Live translation — real-time as-you-type with 500ms debounce
  • 8 target languages — French, Spanish, German, Hindi, Japanese, Chinese, Italian, Telugu
  • Pronunciation / romanization — for Japanese, Hindi, and Telugu scripts
  • Text-to-speech — listen to translations via Web Speech API
  • Share translations — via Web Share API or clipboard fallback
  • Translation history — search, filter, export/import as JSON, favorite items
  • Keyboard shortcuts — Ctrl+Enter, Esc, Ctrl+Shift+C, Ctrl+Shift+S
  • Dark mode — system detection + manual toggle, persisted across sessions
  • Responsive design — optimized for mobile, tablet, and desktop

Architecture

  • Backend: Java 17 + Embedded Jetty 11
  • Caching: Caffeine (10K entries, 60min TTL)
  • Resilience: Resilience4j circuit breakers per provider
  • Metrics: Micrometer + Prometheus (GET /metrics)
  • Health: GET /health + GET /ready
  • Translation chain: Google → LibreTranslate → MyMemory (cascading fallback)
  • Frontend: Vanilla ES Modules — zero build step, zero dependencies

Technologies

  • Java 17, Maven, Embedded Jetty 11, Jakarta Servlets
  • Caffeine cache, Resilience4j, Micrometer, Gson
  • SLF4J + Logback with structured JSON logging
  • Vanilla JS (ES Modules), Fetch API, Web Share API, Web Speech API
  • CSS Grid, Flexbox, CSS Custom Properties, Glassmorphism
  • Docker multi-stage build, GitHub Actions CI/CD

Developer

Built as a portfolio project demonstrating production-grade Java microservice development, multi-provider API fallback chains, circuit breakers, caching, metrics, and modern frontend practices with zero dependencies.
