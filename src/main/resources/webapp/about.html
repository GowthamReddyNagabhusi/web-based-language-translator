About This App

This small translator app performs instant translations using the server-side /translate endpoint. It supports auto-detect for source language (when available) and several target languages.

APIs

  • Primary translation uses a high-quality translate API (configured server-side).
  • Fallbacks include LibreTranslate / MyMemory when available.

Technologies

  • Java + Jakarta Servlets (backend)
  • Vanilla JavaScript (ES Modules)
  • Fetch API + AbortController
  • CSS Grid & Flexbox for layout
